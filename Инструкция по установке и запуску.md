# Система заявок на билеты

Автономное desktop-приложение (Windows). Не требует других программ проекта — только Python и Excel-база сотрудников.

## Установка (Windows)

1. Установите **Python 3.9+** (отметьте «Add to PATH»).
2. Запустите **`install.bat`** в папке программы.
3. Опционально: [Tesseract OCR](https://github.com/UB-Mannheim/tesseract/wiki) с русским языком — для сканированных PDF.

## Запуск

- **`run.bat`** — обычный запуск
- или: `venv\Scripts\python.exe main.py`

## Первый запуск

| Роль | Действие |
|------|----------|
| **Admin** | Войти → «Загрузить базу» → выбрать Excel (лист «ВСЕ ОП» или первый лист с колонкой ФИО) |
| **ОП** | Войти под своим подразделением (база подгрузится из кэша после загрузки Admin) |

Пароли по умолчанию — в `config.py`. Для смены скопируйте `users.local.json.example` → `users.local.json`.

## Рабочий процесс

1. **Каталог** — добавить сотрудников в заявку вручную
2. **PDF файл / Папка PDF** — распознать заявки, заполнить в мастере
3. **Заполнить из базы** — доработать ранее загруженные PDF
4. **Экспорт Excel** — готовый файл заказа билетов
5. Меню **Файл → Черновик** — сохранить/продолжить позже

## Данные программы

Папка `data/` (создаётся автоматически):

| Файл | Назначение |
|------|------------|
| `employees_cache.pkl` | Кэш базы сотрудников |
| `routes.json` | Справочник маршрутов (можно редактировать) |
| `draft_application.pkl` | Черновик заявки |
| `app.log` | Журнал работы |

## Проверка

```bat
venv\Scripts\python.exe check_app.py
```

Или в программе: **Справка → Проверка системы**.

## Требования

- Windows 10+
- Python 3.9+
- Excel-база сотрудников (.xlsx)
- Tesseract — только для PDF-сканов без текстового слоя
