# 🪟 Инструкция по запуску на Windows (без root прав)

## ✅ Требования
- **ОС:** Windows 10/11
- **Python:** 3.9 или новее (с python.org)
- **Node.js:** 18+ (опционально, для Electron UI)
- **Tesseract OCR:** Установлен в `C:\Tesseract-OCR\` (для обработки сканов)

---

## 🚀 Быстрый старт

### 1. Автоматическая настройка
Запустите скрипт настройки (двойной клик или через консоль):
```cmd
windows_setup.bat
```

Скрипт автоматически:
- ✅ Проверит Python и Node.js
- ✅ Создаст виртуальное окружение (`venv`)
- ✅ Установит все зависимости
- ✅ Создаст папку `data/`
- ✅ Скопирует конфиги из примеров

### 2. Настройка безопасности
После выполнения `windows_setup.bat`:

#### a) Сгенерируйте API ключ:
```cmd
python generate_api_key.py
```
Скопируйте вывод в файл `.env`:
```
TICKET_API_KEY=ваш_уникальный_ключ
```

#### b) Захешируйте пароли:
```cmd
python hash_password.py
```
Введите пароль, скопируйте хеш в `users.local.json`:
```json
{
  "users": {
    "Иванов И.И.": "$2b$12$..."
  }
}
```

#### c) Задайте пароль администратора:
В файле `.env`:
```
TICKET_ADMIN_PASSWORD=ВашНадежныйПароль
```

---

## ▶️ Запуск приложения

### Вариант 1: Tkinter UI (рекомендуется)
```cmd
start_ui.bat
```

### Вариант 2: API сервер (фон)
```cmd
start_api.bat
```
API доступен по адресу: http://127.0.0.1:8000  
Документация Swagger: http://127.0.0.1:8000/docs

### Вариант 3: Electron UI (современный интерфейс)
```cmd
start_electron.bat
```

## 🔧 Решение проблем

### Tesseract не найден
Если видите предупреждение:
```
[WARNING] Tesseract OCR не найден...
```
Убедитесь, что файл существует: `C:\Tesseract-OCR\tesseract.exe`

Если установлен в другое место:
1. Отредактируйте `config.py` (строка 46)
2. Или добавьте в PATH Windows

### Ошибки импорта модулей
```cmd
venv\Scripts\activate
pip install -r requirements.txt --upgrade
```

### Порт 8000 занят
Отредактируйте `.env`:
```
TICKET_API_PORT=8001
```

## 📞 Поддержка

При возникновении ошибок:
1. Проверьте логи в `data/app.log`
2. Убедитесь, что все зависимости установлены
3. Проверьте права доступа к папке `data/`

**Приложение полностью бесплатно и работает без прав администратора.**
